add --trim option for generating smaller binaries

In 2024, September 29th, I got up early and turned my computer on. I was excited to see that the legendery pull request was merged.

Here is the link for the pull request: PR: add --trim option for generation smaller binaries

This pull request introduces a new feature of trimming binary size of the compiled Julia code. Note the the feaure is not enabled by default or well-integrated into the compiler. We have now three Julia scripts in the contrib folder of the source code:

  • juliac-buildscript.jl
  • juliac.jl
  • julia-config.jl

These files can be manually downloaded if your are not compiling your own Julia instance from the source code. The juliac.jl script is the main script that is used to compile the Julia code. The juliac-buildscript.jl is a helper script that is used to compile the juliac.jl script. The julia-config.jl script is used to configure the Julia compiler.
