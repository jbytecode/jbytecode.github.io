Step 1 - Generate the decision matrix content

Please enter manually or copy & paste your data. The first row includes criteria names. The first column includes the names of the alternatives. You can separate values using commas or semicolons.

The default directions will set to max for all criteria. The default weights will be equal weights. The decimal separator is a dot (.) for numbers.

Step 2 - Enter directions of optimization for each criterion

Direction of optimization for each criterion. Use comma between directions. Use "max" for maximization and "min" for minimization. Spaces can be used before and after a comma but they are not necessary.

Step 3 - Enter weights

Please select a weighting method. If the weights are predefined then select the Custom Weights option. The default method is Equal Weights.

Step 4 - Select MCDM methods

Please select the MCDM methods to be applied. If more than one method is selected the results will be listed in the same order as the methods are selected.
